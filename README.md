## Installation requirements:
- Latest version of [FNF](https://ninja-muffin24.itch.io/funkin/download/eyJpZCI6NzkyNzc4LCJleHBpcmVzIjoxNzczNjgyODI1fQ%3d%3d.Bgx7WBWTcWEIC4QkjkLTrqj5qrU%3d);
- A good device (mobile or pc, doesn't matter).

## Setup (for dummies):
- Download mod;
- Unzip mod (important);
- Put mod in 'mods' folder;
- If no 'mods' folder, make new folder called 'mods';
- Open Funkin.exe (or FNF on mobile).
- Have fun!

The mod is cross-platform, meaning it works on both PC and mobile devices (Android/iOS), without having to download separate versions.
