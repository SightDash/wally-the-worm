Welcome to Wally the Worm mod! This file contains mod installation requirements, how to set it up and most importantly: Changelogs!

Installation requirements:
- Latest version of [FNF](https://ninja-muffin24.itch.io/funkin/download/eyJpZCI6NzkyNzc4LCJleHBpcmVzIjoxNzczNjgyODI1fQ%3d%3d.Bgx7WBWTcWEIC4QkjkLTrqj5qrU%3d);
- A good device (mobile or pc, doesn't matter).

How to setup (for dummies):
- Download mod;
- Unzip mod (important);
- Put mod in 'mods' folder;
- If no 'mods' folder, make new folder called 'mods';
- Open Funkin.exe (or FNF on mobile).
- Have fun!

The mod *should* work the same on both PC and mobile devices, without having to download separate versions.
